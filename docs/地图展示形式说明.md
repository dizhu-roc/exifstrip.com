# 地图展示形式说明

## 问：地图的展示形式有哪几种，分别是什么样式？

常见地图底图可按「底图类型」分为以下几类：

### 1. 卫星图 (Satellite / Imagery)
- 卫星/航拍影像，真实地表。
- 样式：照片质感，无路网标注（除非叠加图层）。
- 例：Esri World Imagery、Google Satellite。

### 2. 街道图 / 标准图 (Street / Standard / Road)
- 以道路、地名、POI 为主。
- 样式：彩色路网 + 标签，类似 Google 地图默认视图。
- 例：OpenStreetMap 默认、CartoDB Voyager。

### 3. 地形图 (Terrain)
- 强调地形起伏、等高线、山体。
- 样式：有立体感，适合看海拔和地貌。
- 例：OpenTopoMap、Esri World Topography。

### 4. 深色图 (Dark)
- 深色背景、浅色道路与文字。
- 样式：偏「暗色主题」，对眼睛和夜间使用较友好。
- 例：CartoDB Dark Matter。

### 5. 浅色 / 简约图 (Light / Minimal)
- 背景很浅，细节较少。
- 样式：干净、不抢眼，适合做底图或叠加数据。
- 例：CartoDB Positron。

### 6. 混合图 (Hybrid)
- 卫星图 + 道路与地名等矢量标注叠在一起。
- 样式：既有影像又有路名，信息最全。
- 实现方式：卫星底图 + 标注/边界叠加层（如 Esri World Imagery + Reference/World_Boundaries_and_Places）。

### 7. 户外 / 徒步图 (Outdoors / Hiking)
- 面向徒步、骑行，突出小路、等高线、露营地等。
- 样式：类似地形图但更偏户外用途。

### 8. 水系 / 海洋图 (Water / Nautical)
- 强调河流、湖泊、海岸线，有的带航海信息。
- 样式：水体和岸线非常明显。

---

本项目当前使用 **街道图**：OpenStreetMap 标准瓦片，无需 API Key，样式为道路+地名+POI。
